## 📖 About The Project

Offers Management System is a web application developed using **PHP** and **MySQL**.

The system allows users to publish and manage promotional offers from different businesses such as restaurants, cafés, car washes, stores, and other services.

Users can create offers by providing:

- Offer Title
- Business Name
- Description
- Discount Percentage
- Category
- Offer Image

The goal of this project is to provide a simple platform for sharing promotional offers through an easy-to-use web interface.

## ✨ Features

- User Registration

- Add New Offer

## ⚙ Installation

1. Clone the repository

```bash
git clone https://github.com/Yazeed-211/Offers-Management-System.git
```

2. Open XAMPP.

3. Start Apache and MySQL.

4. Import the `offers.sql` database into phpMyAdmin.

5. Place the project folder inside the `htdocs` directory.

6. Open your browser and visit:

```text
http://localhost/Offers-Management-System
```
